# TextToAudio – Offline Android App

Offline Text-to-Speech Android app built using Python, Kivy, and pyttsx3.

## Features
- 100% offline
- Generates real WAV audio
- Voice selection
- Preview & save audio
- No browser, no internet

## How to use
1. Download APK
2. Install on Android
3. Open app
4. Enter text → Generate → Preview → Save

## Tech
- Python
- Kivy
- pyttsx3
